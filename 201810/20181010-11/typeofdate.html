JavaScript——数据类型

    概述

    javaScript共有六种数据类型（ES6又新增了第七种Symbol类型的值，这里不涉及）

  1. 数值（number）：整数和小数
  2. 字符串（string）：文本（比如Hello World）
  3. 布尔值（boolean）:表示真和假（true和false）
  4. undefined:表示未定义或不存在
  5. null:表示空值，即此处的值为空
  6. 对象（object）：各种值组成的集合

    数值，字符串，布尔值这三种是原始数据类型的值

    对象（object）是合成类型的值，一个对象往往是由多种原始数据类型组成

      ——又细分为：
    1. 狭义的对象（object）;
    2. 数组（array）；
    3. 函数（function）

    undefined和null是两个特殊的值

  1. 确定一个值是什么类型

    1. typeof运算符
    2. instanceof
    3. object.prototype.tostring

    instanceof运算符和Object.prototype.toString方法，将在后文介绍。这里介绍typeof运算符。

    typeof 123 // "number"

    typeof '123' // "string"

    typeof false // "boolean"

    typeof undefined // "undefined"

    function f() {} typeof f // "function"

    typeof window // "object"

    typeof {} // "object"

    typeof [] // "object"

    typeof null // "object"

  2. null和undefined

    在if语句中null和undefined都表示为false

    if (!undefined) { console.log('undefined is false'); } // undefined is false if (!null) { console.log('null is false'); } // null is false undefined == null // true

  3. 布尔值（boolean）

    true表示真，false表示假，下列运算符会返回布尔值

    1. 两元逻辑运算符: && (And)，|| (Or)
    2. 前置逻辑运算符： ! (Not)
    3. 相等运算符：===，!==，==，!=
    4. 比较运算符：>,>=,<,<=

    除了以下六个值被转为false，其他值都是为true

    • undefined
    • null
    • false
    • 0
    • NaN
    • ""或''（空字符串）
